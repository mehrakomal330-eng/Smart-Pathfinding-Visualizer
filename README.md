# Smart-Pathfinding-Visualizer
An interactive Pathfinding Visualizer built using Python and Tkinter that demonstrates how Breadth-First Search (BFS) and Depth-First Search (DFS) algorithms work in real-time.

Features
Set Start and End points
Add and remove walls (obstacles)
Drag to reposition start and end nodes
Visualize BFS and DFS step-by-step
Different colors for exploration and final path

Programming Language: Python
GUI Library: Tkinter
Data Structures Used:
Queue (for BFS)
Stack (for DFS)
Set & Dictionary

Buttons:
BFS → Run Breadth-First Search
DFS → Run Depth-First Search
Reset Path → Clears only the path
Clear All → Resets entire grid

Learning Outcomes
Understanding of BFS & DFS algorithms
Practical use of data structures
Experience with GUI development using Tkinter
Visualization of algorithm behavior
